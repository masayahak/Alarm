■ 通知の仕様
──────────────────────────────────────────────────────────
・予定時刻の30分前に通知が作成される。
　　朝食前のお薬 ＸＸＸＸＸＸＸｘ
　　XX:XXにアラームが鳴ります。
　　　　「お薬を飲み終えた。」
・このタイミングでアプリアイコンの右上に黄色丸がつく。
・この通知が表示されているタイミングでウィジェットをクリックすると
　「朝食前のお薬 ＸＸＸＸＸＸＸＸＸを飲みましたか？」と確認され
　　「お薬を飲み終えた。」をタップできる。
・飲み終えるか、予定時刻になると上記通知は自動的に削除される。
・予定時刻になるとアプリアイコンの右上に赤色！がつく。
・アラームONだと、予定時刻にアプリが起動しアラームが鳴る。
　スヌーズでアラームOFF
　「朝食前のお薬 ＸＸＸＸＸＸＸＸＸを飲みましたか？」と確認され
 　　「お薬を飲み終えた。」をタップできる。
