■ 通知の仕様
──────────────────────────────────────────────────────────
・予定時刻の30分前に通知が作成される。
　　朝食前のお薬 ＸＸＸＸＸＸＸｘ
　　XX:XXにアラームが鳴ります。
　　　　「お薬を飲み終えた。」
・このタイミングでアプリアイコンの右上に丸がつく。
・この通知が表示されているタイミングでアイコンをクリックすると
　「朝食前のお薬 ＸＸＸＸＸＸＸＸＸを飲みましたか？」と確認され
　　「お薬を飲み終えた。」をタップできる。
・飲み終えるか、予定時刻になると上記通知は自動的に削除される。
・アラームONで、予定時刻に飲み終えてないとアプリが起動しアラームが鳴る。
　スヌーズでアラームOFF
　「朝食前のお薬 ＸＸＸＸＸＸＸＸＸを飲みましたか？」と確認され
 　　「お薬を飲み終えた。」
    「30分後にもう一度アラームを鳴らす」
    をタップできる。

テストアプリの仕様
──────────────────────────────────────────────────────────
・２種類の通知を発行できる。
【通知１：３０分前の挙動】
・ボタンクリックでアプリ終了。5秒後に通知が作成される。
・通知にはボタンがあり、ボタンを押すと任意の処理が実行できる。（暫定的にTOAST）
　さらに、ボタンを押すと通知は消える。
【通知２：予定時刻のの挙動】
・ボタンクリックでアプリ終了。10秒後にアラーム画面でアプリが自動起動する。
・アラーム画面は音がなり、スヌーズでアラームをOFFにできる。
